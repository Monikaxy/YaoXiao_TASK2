RSI、MACD、布林带指标介绍

RSI（相对强弱指标）是用来衡量股票一段时间内上涨和下跌强弱的技术指标，可以帮助判断股价是否处于超买或超卖状态。它通常以 14 个交易日为计算周期，先计算平均上涨幅度和平均下跌幅度，再求出两者的比值 RS，最后利用公式 RSI = 100 - 100 / (1 + RS) 得到结果。一般认为 RSI 大于 70 时，股票可能涨得过多，存在回调风险；RSI 小于 30 时，股票可能跌得较多，后续可能出现反弹。

MACD（指数平滑异同移动平均线）主要通过比较短期和长期指数移动平均线 EMA 的差值来分析股价走势。计算方法是先计算 12 日 EMA 和 26 日 EMA，两者相减得到 DIF，再计算 DIF 的 9 日 EMA 得到 DEA，最后用 MACD = 2 × (DIF - DEA) 得到柱状图。当 DIF 从下向上突破 DEA 时称为金叉，一般认为是买入信号；当 DIF 从上向下跌破 DEA 时称为死叉，一般认为是卖出信号。

布林带（Bollinger Bands）由上轨、中轨和下轨三条线组成，通常以 20 日移动平均线作为中轨，再利用股价 20 日标准差计算上下轨，其中上轨等于中轨加 2 倍标准差，下轨等于中轨减 2 倍标准差。布林带可以帮助判断股价是否偏高或偏低，也可以观察市场波动情况。

Python计算过程

整体计算流程是：读取 CSV，将交易日期转换为日期格式并升序排列；使用收盘价计算 RSI、MACD 和布林带；再使用最高价、最低价和收盘价计算 KDJ；最后保存新的结果表，并用网页中的交互图展示。

import numpy as np
import pandas as pd

df = pd.read_csv("贵州茅台_2025-06_2026-06_日交易数据.csv", encoding="utf-8-sig")
df["交易日期"] = pd.to_datetime(df["交易日期"].astype(str), format="%Y%m%d")
df = df.sort_values("交易日期").reset_index(drop=True)

for col in ["最高价", "最低价", "收盘价"]:
    df[col] = pd.to_numeric(df[col], errors="coerce")

# RSI(14)
delta = df["收盘价"].diff()
gain = delta.clip(lower=0)
loss = -delta.clip(upper=0)
avg_gain = gain.ewm(alpha=1 / 14, adjust=False, min_periods=14).mean()
avg_loss = loss.ewm(alpha=1 / 14, adjust=False, min_periods=14).mean()
rs = avg_gain / avg_loss
df["RSI_14"] = 100 - 100 / (1 + rs)

# MACD(12,26,9)
df["EMA12"] = df["收盘价"].ewm(span=12, adjust=False).mean()
df["EMA26"] = df["收盘价"].ewm(span=26, adjust=False).mean()
df["DIF"] = df["EMA12"] - df["EMA26"]
df["DEA"] = df["DIF"].ewm(span=9, adjust=False).mean()
df["MACD"] = 2 * (df["DIF"] - df["DEA"])

# 布林带(20,2)
df["BB_MID"] = df["收盘价"].rolling(window=20).mean()
df["BB_STD"] = df["收盘价"].rolling(window=20).std()
df["BB_UPPER"] = df["BB_MID"] + 2 * df["BB_STD"]
df["BB_LOWER"] = df["BB_MID"] - 2 * df["BB_STD"]

# KDJ(9)
low_9 = df["最低价"].rolling(window=9, min_periods=9).min()
high_9 = df["最高价"].rolling(window=9, min_periods=9).max()
df["RSV_9"] = (df["收盘价"] - low_9) / (high_9 - low_9) * 100
df["K"] = np.nan
df["D"] = np.nan

first_valid_index = df["RSV_9"].first_valid_index()
if first_valid_index is not None:
    previous_k = 50.0
    previous_d = 50.0
    for index in range(first_valid_index, len(df)):
        rsv = df.loc[index, "RSV_9"]
        df.loc[index, "K"] = previous_k * 2 / 3 + rsv / 3
        df.loc[index, "D"] = previous_d * 2 / 3 + df.loc[index, "K"] / 3
        previous_k = df.loc[index, "K"]
        previous_d = df.loc[index, "D"]

df["J"] = 3 * df["K"] - 2 * df["D"]
df.to_csv("贵州茅台_技术指标完整结果.csv", index=False, encoding="utf-8-sig")

扩展指标KDJ介绍、计算和可视化

KDJ 是一个常见的短线技术指标，主要用来判断股价短期强弱和超买超卖情况。它把最高价、最低价和收盘价结合起来，看当前收盘价在最近一段价格区间里的位置。本次使用 9 日 KDJ，先计算 RSV = (收盘价 - 9日最低价) / (9日最高价 - 9日最低价) × 100，再递推得到 K 值和 D 值，最后 J = 3 × K - 2 × D。一般 KDJ 高于 80 附近表示可能偏超买，低于 20 附近表示可能偏超卖。

KDJ 指标

分析：KDJ 的 J 值波动最大，说明它对短期价格变化比较敏感；后期 K、D 位于较低区域，说明短线情绪偏弱，但也可能出现技术性反弹。
